{\bfseries{\mbox{\hyperlink{namespaceAirControl}{Air\+Control}} is an Open Source, Modular, Cross-\/\+Platform, and Extensible Flight Simulator For Deep Learning Research.}} Airsim offers a realistic simulation experience with a variety of airplanes. The Airsim is built on \href{https://unity.com}{\texttt{ Unity Game engine}}. Following are the salient features of the Aircontrol\+:


\begin{DoxyItemize}
\item Built with {\bfseries{C\#}}, it has {\bfseries{Python}} A\+PI to control it from your favorite Deep learning Framework.
\item Complete source code is open on Github.
\item Aircontrol takes full advantage of object-\/oriented programming. Developed fully modular from day one. You can easily introduce new features such as {\bfseries{vertical takeoff}}. you can bring your own {\bfseries{alien plane to Air\+Cotrol}}.
\item \mbox{\hyperlink{namespaceAirControl}{Air\+Control}} is truly cross-\/platform, can be compiled on Linux, mac\+OS, and Windows. Binary will be released for all the platforms.
\item The \mbox{\hyperlink{namespaceAirControl}{Air\+Control}} uses Nvidia \href{https://en.wikipedia.org/wiki/PhysX}{\texttt{ Physx}} for the best possible Newtonian physics simulation.
\item Aircontrol allows users to take advantage of aerodynamics effects such as \href{https://en.wikipedia.org/wiki/Ground_effect_(aerodynamics)}{\texttt{ Ground effect}}.
\item All the control surfaces (Throttle, Rudder, Ailerons, and Flaps) accepts normalized input between -\/1 and 1. This makes Aircontrol even more friendly with AI.
