v2.3.0

版本要点

  • 数据存储由float升级为double
  • 新增MarkLine标线
  • Serie下可用IconStyle统一配置图标
  • Label支持用代码自定义显示样式
  • DataZoom完善
  • PieChart优化
  • 问题修复

升级注意

由于数据类型升级为了double，float隐式转double可能有精度问题，所以建议之前为float的数据类型都手动改为double类型。
